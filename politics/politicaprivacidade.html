Politica de Privacidade - App Jesus Today

Política de coleta de dados

1.0 O aplicativo Jesus Today recolhe o email do usuário que se cadastrar no app.
1.1 O uso do aplicativo Jesus Today não requer o cadastro para o uso pleno do aplicativo, no entanto utilizamos o email do usuário para salvar o progresso dele dentro do app.
1.2 Utilizamos bancos de dados externos do google, providos pelo firebase, para armazenarmos o email, senha e progresso de leitura do app.
1.3 Ao clicar em compartilhar, seja o versículo do dia ou qualquer outro versículo da bíblia, a imagem selecionada para compartilhar é salva automaticamente no dispositivo do usuário para o compartilhamento.
1.4 Quando o usuário grifar um versículo os dados são salvos automaticamente no dispositivo do usuário em um banco de dados interno, no entanto nada é armazenado remotamente.
